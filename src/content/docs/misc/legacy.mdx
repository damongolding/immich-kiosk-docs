---
title: Legacy browser support
---

The main Kiosk build only supports ([relatively](https://docs.immichkiosk.app/#requirements)) newer browsers.

If you are trying to run Kiosk on old browsers you can give the legacy experiment I created a try. I would suggest trying to load the [demo](https://legacy.immichkiosk.app) on the device as a test before using the docker image.

Here is a link to the [Github repo](https://github.com/damongolding/immich-kiosk-legacy)


## Demo
A demo instance of legacy [can be viewed here.](https://legacy.immichkiosk.app) 

The legacy demo points at the demo instance of Kiosk (https://demo.immichkiosk.app).
