## Disable UI

A shortcut to set show_time, show_date, show_image_time and image_date_format to false.

<Settings
    config="disable_ui"
    env="KIOSK_DISABLE_UI"
    url="disable_ui"
    value="bool"
    defaultValue="false"
/>

## Frameless

Remove borders and rounded corners on images.

<Settings
    config="frameless"
    env="KIOSK_FRAMELESS"
    url="frameless"
    value="bool"
    defaultValue="false"
/>

## Hide Cursor

Hide cursor/mouse via CSS.

<Settings
    config="hide_cursor"

    defaultValue="false"

## Font Size

The base font size for Kiosk. Default is 100% (16px). DO NOT include the % character.

<Settings
    config="font_size"
    env="KIOSK_FONT_SIZE"
    url="font_size"
    value="int"
    defaultValue="100"
/>


## Background Blur

Display a blurred version of the image as a background.

<Settings
    config="background_blur"
    env="KIOSK_BACKGROUND_BLUR"
    url="background_blur"
    value="bool"
    defaultValue="true"
/>

<Settings
