<Aside type="caution">
    Not all filters work on all asset source/buckets.
</Aside>

Filters allow you to filter asset buckets (people/albums/date etc.) by certain criteria.


## Date filter

Limit assets from sources to a given date range.

`filter_date` accepts the same values as [date range](/configuration/date-range).

<Settings
    config="filter_date"
    env="KIOSK_FILTER_DATE"
    url="filter_date"
    value="string"
    urlOver={"http://{URL}?person=PERSON_ID&filter_date=2023-01-01_to_2023-02-01"}    
/>

### Examples

- `http://{URL}?filter_date=last-30-days` will only show (random) assets from the last 30 days.
- `http://{URL}?filter_date=2021-01-01_to_today` will only show (random) assets between 2021-01-01 and today.

## Favorite filter

Limit asset sources to only favorite assets.

<Settings
    config="filter_favorites"
    env="KIOSK_FILTER_FAVORITES"
    url="filter_favorites"
    value="bool"
    defaultValue="false"
    configOver={`filter_favorites: true`}
    envOver={`environment:
    KIOSK_FILTER_FAVORITES: true
    `}
    urlOver={"http://{URL}?album=ALBUM_ID&filter_favorites=true"}
/>

### Examples

- `http://{URL}?filter_favorites=true` will only show favourite random assets.
- `http://{URL}?album=ALBUM_ID&filter_favorites=true` will only show favourite assets from the supplied album.

## Newest filter

Limit asset sources to only the newest X assets.


<Settings
    config="filter_newest"
    env="KIOSK_FILTER_NEWEST"
    url="filter_newest"
    value="int"
/>

### Examples

- `http://{URL}?filter_newest=200` will only show (random) assets from the newest 200.
- `http://{URL}?album=XXX&filter_newest=200` will only show the newest 200 (random) assets from supplied album.


## Exclude faces filter

<Aside type="note">
    Requires Immich to have already processed face detection on your assets.
</Aside>

Excludes assets where Immich has detected a face. Useful for slideshows focused on scenery, landscapes, or architecture.

<Settings
    config="filter_exclude_faces"
    env="KIOSK_FILTER_EXCLUDE_FACES"
    url="filter_exclude_faces"
    value="bool"
    defaultValue="false"
/>
