## Frame padding

Add padding around the outside of Kiosk.

<Settings
    config="frame_padding"
    env="KIOSK_FRAME_PADDING"
    url="frame_padding"
    value="int | []int"
    defaultValue="0"
/>

### Frame padding examples

#### Single padding
```yaml
frame_padding: 10 ## add 10px padding around the outside of Kiosk
```

#### Top and bottom padding
```yaml
frame_padding: [10, 20] ## add 10px padding top and bottom, 20px padding right and left
```

#### Top, right, left and bottom padding
```yaml
frame_padding: [10, 20, 30, 40] ## add 10px padding top, 20px padding right, 30px padding bottom, 40px padding left
```

#### URL params
##### Add 10px top and bottom and 20px left and right padding via URL
```http
https://example.com?frame_padding=10&frame_padding=20
```

##### Add 10px top, 20px right, 30px bottom and 40px left padding via URL
```http
https://example.com?frame_padding=10&frame_padding=20&frame_padding=30&frame_padding=40
```
